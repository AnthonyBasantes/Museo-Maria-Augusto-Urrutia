# Proyecto: Sistema Web - Museo Maria Augusta Urrutia 🏛️

Este proyecto es una aplicación web dinámica desarrollada para la gestión y visualización de información del **Museo Maria Augusta Urrutia**. El sistema permite navegar por las galerías del museo y gestionar contenidos mediante tecnologías Java empresariales.

## 🚀 Tecnologías Utilizadas

* **Backend:** Java Server Pages (JSP) y Servlets.
* **Frontend:** HTML5, CSS3 y JavaScript para una interfaz interactiva.
* **Gestión de Base de Datos:** PostgreSQL.
* **Servidor de Aplicaciones:** Apache Tomcat.
* **Control de Versiones:** Git y GitHub.

## 📂 Estructura del Proyecto

El repositorio contiene la estructura estándar de una aplicación web Java:
* `src/main/webapp`: Contiene las páginas `.jsp`, hojas de estilo `css` e imágenes del sitio.
* `.settings` y `.project`: Archivos de configuración del entorno de desarrollo (Eclipse/IDE).
